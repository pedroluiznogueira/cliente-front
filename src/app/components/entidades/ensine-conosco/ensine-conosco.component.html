    Este curso vai te mostrar como centralizar elementos usando css puro e flex-box

    Primeiro vamos preparar os elementos que queremos centralizar e manipular. Para esse tutorial, usaremos uma div sendo um container pai que engloba outras três divs sendo os filhos desse container pai.

    Quando trabalhamos com flex-box este conceito é muito importante, sempre que quisermos manipular ou aplicar qualquer funcionalidade do flex, precisamos aplicá-las pelo css do container pai que engloba os filhos que receberão estas funcionalidades

    Agora vamos aplicar algumas propriedades do css para deixar os elementos esplícitos visualmente falando, de forma que fique mais fácil de vizualizar o que está acontecendo.

    Para começarmos à utilizar flex box, nós fazemos o seguinte:

    Ao aplicarmos o "display: flex" dizemos que os filhos receberão as propriedades do flex

    "justify-content: center" centraliza os filhos na horizontal

    "align-items: center" centraliza os filhos na vertical
